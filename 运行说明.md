# 运行说明

## 方式一：使用Maven Tomcat插件（推荐）

项目已经配置了Tomcat Maven插件，可以直接运行：

```bash
cd /Users/nullaura/Desktop/web-final
mvn tomcat7:run
```

服务器启动后，访问：**http://localhost:8080/linux-desktop/**

## 方式二：部署到独立的Tomcat服务器

1. 确保已经编译项目：
```bash
mvn clean package
```

2. 将生成的 `target/linux-desktop.war` 复制到Tomcat的 `webapps` 目录

3. 启动Tomcat服务器

4. 访问：**http://localhost:8080/linux-desktop/**

## 使用步骤

1. **连接SSH服务器**
   - 打开浏览器访问应用
   - 点击任务栏的"SSH连接"按钮
   - 输入Linux服务器信息：
     - 主机地址（如：192.168.1.100 或 localhost）
     - 端口（默认22）
     - 用户名
     - 密码
   - 点击"连接"

2. **使用文件管理器**
   - 点击"文件管理器"按钮
   - 双击文件夹进入目录
   - 双击文件打开查看（文本文件）
   - 右键文件选择"属性"查看详细信息

3. **使用终端**
   - 点击"终端"按钮
   - 输入Linux命令执行

4. **查看进程**
   - 点击"进程管理"按钮
   - 查看系统所有运行进程

## 注意事项

- 确保Linux服务器的SSH服务已启动
- 确保防火墙允许SSH连接（端口22）
- 如果连接失败，请检查用户名和密码是否正确
